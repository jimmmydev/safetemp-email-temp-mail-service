SafeTemp Email is designed with a core principle: preserving user privacy without compromise.

## 1. No Data Collection
The service does not store:
- Personal data
- IP addresses
- Tracking cookies
- Persistent identifiers
- Browsing information

## 2. Temporary Emails
All emails are processed only for functional purposes.  
When a mailbox expires, the information is deleted immediately.

## 3. Ephemeral Sessions
Each session is isolated and not linked to any real identity.

## 4. No Analysis or Commercialization
SafeTemp Email does not sell, share, or analyze user data with third parties.

## 5. Incoming Communications
Emails exist only while active in the temporary inbox.  
Once expired, they are permanently deleted.

## 6. Security
The backend includes:
- Anti-abuse filters
- Traffic limiters
- Automated cleanup
- Isolation modules

## 7. Future Updates
If the API or advanced features are added in the future, this policy will be updated to reflect functionality while maintaining privacy.
